ゲーム説明
******************************************************************
ビリヤードの9ボールのルールを反転させたゲーム
　
ルール
　・数字のついているボールを打って白いボールのみを落とせ！
　・打つときは必ずどれかのボールに当てないとGameOver
　　（当てるボールは何番でもOK）
　・白いボールを最初に落としたらゲームクリア！


操作
メニュー
　エンターキー　　　　決定
　十字キー　　　　　　選択
　Zキー　　　　　　　 操作説明表示：非表示

ゲーム中
　スペースキー　	　球を打つ
　左キー、Aキー　　　打つ方向を変更（左回り）
　右キー、Dキー　　　打つ方向を変更（右回り）
　Xキー　　　　　　　ボール番号、表示・非表示
　Zキー	  	    操作説明表示：非表示
   Qキー、ESCキー　　ゲーム終了
